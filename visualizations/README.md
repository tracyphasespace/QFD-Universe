# QFD Interactive Visualizations

Interactive HTML visualizations for Quantum Field Dynamics concepts. **No installation required** - just click the links below to view in your browser.

**[View Visualization](https://tracyphasespace.github.io/QFD-Universe/visualizations/SolitonCanonPhoton2.HTML)**

Interactive 3D visualization of the driven nonlinear oscillator model showing:
- Photon soliton dynamics in real-time
- Phase space trajectories
- Energy transfer mechanisms
- Uses Three.js for GPU-accelerated rendering

### 2. Vortex Dynamics
**[View Visualization](https://tracyphasespace.github.io/QFD-Universe/visualizations/vortex_dynamics_visualization.html)**

Animated visualization of QFD vortex field behavior:

- Field line evolution

### 3. Duffing Oscillator
**[View Visualization](https://tracyphasespace.github.io/QFD-Universe/visualizations/DuffingOscillator.html)**

Classic nonlinear oscillator demonstrating:
- Bifurcation dynamics

### Option 1: GitHub Pages (Recommended)

2. Open any `.html` file in a modern browser
3. No server required - works offline
