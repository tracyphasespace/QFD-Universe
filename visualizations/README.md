# QFD Interactive Visualizations

Interactive HTML visualizations for Quantum Field Dynamics concepts. **No installation required** - just click the link below to view in your browser.

---

## Available Visualizations

### Photon Soliton Canon
**[View Visualization](https://tracyphasespace.github.io/QFD-Universe/visualizations/PhotonSolitonCanon.html)**

Interactive 3D visualization of photon soliton dynamics showing:
- Real-time photon field evolution
- Phase space trajectories
- Energy transfer mechanisms
- Driven nonlinear oscillator behavior

Uses Three.js for GPU-accelerated WebGL rendering.

---

## How to View

### Option 1: GitHub Pages (Recommended)
Click the link above to view directly in your browser.

### Option 2: Local Viewing
1. Clone the repository
2. Open `PhotonSolitonCanon.html` in a modern browser
3. No server required - works offline

3. Update this README with description and link
4. Submit pull request
